A `date` field can be formatted using the `{{format_date}}` helper. Example markup using a `date` field with the variable name `date`:
```
<p>{{format_date fields.date "D" "us"}}</p>
```

Example Output:
```
<p>Tuesday, May 16, 2017</p>
```
