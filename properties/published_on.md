This is the timestamp of when the current page or item was published. The publish date can be set in the Admin panel. It can be formatted using the `{{format_date}}` helper.

Example Markup:
```
<p>{{format_date published_on "d" "us"}}</p>
```

Example Output:  
```
<p>5/17/2017</p>
```
